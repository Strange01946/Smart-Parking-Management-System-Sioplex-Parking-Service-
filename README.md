# Smart Parking Management System (Sioplex Parking Service)

## Overview
This project is a GUI-based Parking Management System developed using Python and Tkinter. The system manages vehicle parking operations including slot allocation, vehicle tracking, billing, and parking lot visualization.

The application supports multiple vehicle types and provides real-time parking status along with revenue tracking and vehicle history management.

## Features
- Admin login authentication system
- Multi-vehicle parking support (Car, Bike, Truck, Bus)
- Automatic parking slot allocation
- Emergency parking slot handling
- Real-time parking status visualization
- Vehicle search functionality
- Parking fee calculation based on duration
- Digital receipt generation
- Exit history tracking
- Total revenue reporting
- Vehicle blacklist management
- Data persistence using JSON files

## Technologies Used
- Python
- Tkinter (GUI)
- JSON (Data Storage)
- Datetime module

## System Components
- Admin authentication system
- Parking slot management module
- Vehicle search and tracking system
- Revenue and billing system
- Exit history management
- Blacklist vehicle management

## How It Works
1. Admin logs into the system.
2. Vehicles are parked with automatic slot allocation.
3. Parking data is stored and updated in JSON files.
4. When a vehicle exits, parking duration and fees are calculated.
5. The system generates a digital receipt and updates revenue records.
